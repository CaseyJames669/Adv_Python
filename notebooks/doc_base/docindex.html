including
    document name and contents,
    list of unique words in the document, and

class DocIndex(builtins.object)
    DocIndex(document, title) -> initialized object for one document
 
Manage an index of words occurring in a text document, including
    document name and lines of the document.
Can report
    document name
    list of unique words
    contexts for words

get_context(self, term, lines=1)
Return a generator over occurrences of the term in context.
 
The generator will provide a stream of "contexts" for the specified
    (apparent) word in 'term'.
 A "context" displays the line containing the word, along with  
    the number of preceeding and succeeding lines specified in the 
    parameter 'lines' (if they exist) ... defaults to 1
    
Preconditions:
    term is a string containing the word to look up
 
Postcondition:
    a generator is created to deliver the stream of contexts
get_label(self)
Return the string which serves as the document name.

Return the document text
get_words(self)
Return an ordered list of words appearing in this document.
